使用Ubuntu 配置本地git 并同步github远程仓库

2021-11-01

apt install git
git config --global user.email "填入邮箱"
git config --global user.name "填入名称"

ssh-keygen -t rsa
cat /root/.ssh/id_rsa.pub
打开之后复制，然后粘贴到git hub 远程仓库的SSH keys 中

输入下面的命令检查是否成功哦那个配置ssh
ssh -T git@github.com

使用git clone从远程仓库下载文件
git clone 这里输入SSH的那个链接

使用git push上传文件到远程仓库
git add xxx
查看当前git状态
git status
commit 文件
git commit -m "输入备注"
上传文件
git push origin master

使用git remote -v查看远程仓库地址
使用git remote -vv查看当前工作branch
使用git remote add origin 这里输入SSH的那个链接添加 仓库地址
