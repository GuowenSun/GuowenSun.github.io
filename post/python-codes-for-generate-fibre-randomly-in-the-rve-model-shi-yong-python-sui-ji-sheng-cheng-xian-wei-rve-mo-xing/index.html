Python codes for generate fibre randomly in the RVE model 使用Python随机生成纤维 RVE 模型

2021-08-05

The initial concept is to build a RVE model with the fibre centre randomly generated within a specified boundary. In this blog, my intial concept will be presented. Although it provides a very low effiency and low productivity on the fibre generation, it has a big potential for being a powerful RVE model generator with random fibre distribution.

The RVE means the representative volume element that is sufficient to represent effective properties of composites.
Since we need to generate a RVE model, there are several parameters requiring to be defined. The essential one is the fibre radius. The fibre radius value can be obtained from the SEM observation with adoption of image processing. Such process I achieved is by the Python. In this example, the fibre radius of 0.38679 was adopted.
The boundary of fibre generating area should be defined further. It should be 50 times of fibre radius value according to Trias work done.
